## Open Graph Metatags

```html
<!doctype html>
<html lang="ru_RU">
<head>
    <meta charset="UTF-8">
		
    <meta name="description" content="Помимо информации на странице, которая видна пользователю и расположена в теге body на вебсайте есть техническая информация, которая размещена в метатегах элемента head. В каждой мета-директиве мы можем повлиять на один или несколько парамтров, которые считываются браузером, поисковиком, социальной сетью или другими сайтами.  " />
    <meta property="og:locale" content="ru_RU" />
    <meta property="og:type" content="article" />
    <meta property="og:title" content="Что такое метатеги, на что они влияют и как их правильно настраивать" />
    <meta property="og:description" content="Помимо информации на странице, которая видна пользователю и расположена в теге body на вебсайте есть техническая информация, которая размещена в метатегах элемента head. В каждой мета-директиве мы можем повлиять на один или несколько парамтров, которые считываются браузером, поисковиком, социальной сетью или другими сайтами.  " />
    <meta property="og:url" content="https://andron13.de/content/article-179" />
    <meta property="og:site_name" content="ANDRON13 - бэкенд, фронтенд, Берлин" />
    <meta property="article:published_time" content="2021-09-26T09:38:46+00:00" />
    <meta property="article:modified_time" content="2021-09-26T09:57:43+00:00" />
    <meta property="og:image" content="https://andron13.de/content/article-179" />
    <meta property="og:image:width" content="640" />
    <meta property="og:image:height" content="480" />
    <meta name="twitter:card" content="summary_large_image" />
    <meta name="twitter:label1" content="Написано автором">
    <meta name="twitter:data1" content="Андрей Подлубный">
    
    

    <meta name="robots" content="index, follow, max-snippet:-1, max-image-preview:large, max-video-preview:-1" />
    <link rel="canonical" href="https://andron13.de/content/article-179" />
    <meta name="viewport"
          content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<body>
<!--Контент сайта-->
</body>
</html>
```
